JavaScript adalah bahasa pemrograman yang digunakan untuk membuat website menjadi interaktif. Jika HTML adalah struktur dan CSS adalah desain, maka JavaScript adalah bagian yang membuat website bisa bergerak, merespon klik, menampilkan pesan, memvalidasi form, dan masih banyak lagi.

Kesimpulan: JavaScript membuat website terasa "hidup" dan interaktif.

JavaScript digunakan untuk berbagai hal seperti:

  • Membuat tombol bisa melakukan aksi ketika diklik
  • Membuat validasi login dan form
  • Membuat animasi sederhana
  • Mengubah isi halaman tanpa reload (dynamic content)
  • Mengambil data dari server (API)
  • Membuat game atau aplikasi berbasis web

ðŸ“Œ Variabel

Variabel digunakan untuk menyimpan data. Di JavaScript kita bisa menggunakan var, let, dan const.

JavaScript memiliki beberapa tipe data utama:

ðŸ“Œ Operator

Operator digunakan untuk melakukan perhitungan dan logika.

Operator Aritmatika: 
let a = 10;
let b = 5;

console.log(a + b); // 15
console.log(a - b); // 5
console.log(a * b); // 50
console.log(a / b); // 2
Operator Perbandingan: 
console.log(10 > 5);   // true
console.log(10 == 5);  // false
console.log(10 != 5);  // true

Percabangan digunakan untuk membuat keputusan.

ðŸ“Œ Perulangan (Looping)

Looping digunakan untuk mengulang kode.

Contoh perulangan for: 
for (let i = 1; i <= 5; i++) {
  console.log("Perulangan ke-" + i);
}
Contoh perulangan while: 
let i = 1;

while (i <= 3) {
  console.log("Halo ke-" + i);
  i++;
}

ðŸ“Œ Event (Klik Tombol)

Event digunakan untuk menjalankan kode saat terjadi aksi tertentu, misalnya klik.

ðŸ“Œ DOM (Document Object Model)

DOM adalah cara JavaScript mengakses dan memanipulasi elemen HTML. Dengan DOM kita bisa mengubah teks, warna, atau isi halaman secara langsung.

Contoh mengubah teks: 
document.getElementById("judul").innerHTML = "Judul Baru!";

Contoh lengkap:

<h2 id="judul">Halo Dunia</h2>
<button onclick="ubahJudul()">Ubah Judul</button>

<script>
  function ubahJudul() {
    document.getElementById("judul").innerHTML = "Belajar JavaScript Itu Seru!";
  }
</script>
