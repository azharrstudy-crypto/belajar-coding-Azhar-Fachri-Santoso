ðŸ“Œ Apa Itu HTML?

HTML adalah singkatan dari HyperText Markup Language. HTML merupakan bahasa markup yang digunakan untuk membuat struktur halaman website. HTML bukan bahasa pemrograman, tetapi bahasa untuk menyusun elemen-elemen website seperti teks, gambar, link, tabel, form, dan lain-lain.

Kesimpulan: HTML itu ibarat kerangka atau tulang dari sebuah website.

ðŸ“Œ Fungsi HTML

Berikut beberapa fungsi utama HTML:

  • Membuat struktur halaman web
  • Menampilkan teks dan paragraf
  • Menambahkan gambar, audio, video
  • Membuat link (hyperlink)
  • Membuat tabel
  • Membuat form login/daftar
  • Menghubungkan website dengan CSS dan JavaScript

ðŸ“Œ Struktur Dasar HTML

Berikut contoh struktur dasar HTML:

<!DOCTYPE html>
<html>
<head>
  <title>Judul Website</title>
</head>

<body>
  <h1>Halo Dunia!</h1>
  <p>Ini adalah website pertama saya.</p>
</body>
</html>
Penjelasan singkat:
  • <!DOCTYPE html> = menandakan dokumen HTML5
  • <head> = bagian informasi (judul, link CSS, meta)
  • <body> = bagian isi halaman yang terlihat

ðŸ“Œ Tag dan Elemen HTML

Tag HTML adalah kode yang digunakan untuk membuat elemen di website. Tag biasanya ditulis menggunakan tanda < >.

Contoh tag HTML:

  • <h1> untuk judul
  • <p> untuk paragraf
  • <img> untuk gambar
  • <a> untuk link

ðŸ“Œ Heading (Judul)

Heading digunakan untuk membuat judul pada website. Heading memiliki level dari <h1> sampai <h6>. 

<h1>Judul Utama</h1>
<h2>Sub Judul</h2>
<h3>Judul kecil</h3>

ðŸ“Œ Paragraf

Paragraf dibuat menggunakan tag <p>.

<p>Ini adalah paragraf pertama.</p>
<p>Ini adalah paragraf kedua.</p>

ðŸ“Œ Link (Hyperlink)

Link digunakan untuk berpindah halaman atau membuka website lain. Tag link menggunakan <a>. 

<a href="https://google.com">Buka Google</a>
Catatan: Atribut href berisi alamat tujuan link.

ðŸ“Œ Gambar

Untuk menampilkan gambar gunakan tag <img>. Tag ini tidak memiliki penutup. 

<img src="gambar.jpg" alt="Ini gambar">

Keterangan atribut:

  • src = lokasi file gambar
  • alt = teks alternatif jika gambar gagal muncul

ðŸ“Œ List (Daftar)

Ada 2 jenis list dalam HTML:

1. Ordered List (List berurutan) 
<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>
2. Unordered List (List tidak berurutan) 
<ul>
  <li>Belajar HTML</li>
  <li>Belajar CSS</li>
  <li>Belajar JavaScript</li>
</ul>

ðŸ“Œ Tabel

HTML juga bisa digunakan untuk membuat tabel.

<table border="1">
  <tr>
    <th>Nama</th>
    <th>Kelas</th>
  </tr>
  <tr>
    <td>Azhar</td>
    <td>XI RPL</td>
  </tr>
</table>

ðŸ“Œ Form (Input Data)

Form digunakan untuk input data seperti login, daftar, search, dll. 

<form>
  <label>Username:</label>
  <input type="text">

  <label>Password:</label>
  <input type="password">

  <button type="submit">Login</button>
</form>

ðŸ“Œ HTML dan CSS

HTML digunakan untuk membuat struktur halaman. Sedangkan CSS digunakan untuk mempercantik tampilannya. Jadi HTML dan CSS selalu bekerja bersama dalam pembuatan website.

Contoh:

HTML = membuat tombol
CSS = membuat tombol jadi bagus, berwarna, dan animasi hover

ðŸ“Œ HTML dan JavaScript

Jika HTML adalah kerangka dan CSS adalah desain, maka JavaScript adalah "otak" dari website. JavaScript membuat website menjadi interaktif seperti tombol bisa diklik, validasi login, popup, dll.

ðŸ“Œ Kesimpulan

HTML adalah bahasa dasar yang wajib dipelajari dalam pembuatan website. Dengan HTML, kita bisa membuat struktur halaman web seperti judul, paragraf, gambar, tabel, link, dan form. Setelah menguasai HTML, langkah selanjutnya adalah mempelajari CSS dan JavaScript agar website menjadi lebih menarik dan interaktif.
