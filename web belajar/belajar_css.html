CSS adalah singkatan dari Cascading Style Sheets. CSS merupakan bahasa yang digunakan untuk mengatur tampilan halaman website, seperti warna, ukuran teks, font, jarak, layout, dan desain lainnya.

Kesimpulan: HTML itu kerangka website, sedangkan CSS itu yang membuat website jadi cantik.

âš™ Fungsi CSS

CSS memiliki beberapa fungsi penting, antara lain:

  • Mengubah warna teks dan background
  • Mengatur ukuran font dan jenis font
  • Membuat layout website lebih rapi
  • Membuat tombol, menu, dan tampilan website lebih menarik
  • Membuat website responsif di berbagai ukuran layar

ðŸ“Œ Cara Kerja CSS

CSS bekerja dengan cara memilih elemen HTML, lalu memberikan aturan (style) pada elemen tersebut. Aturan ini ditulis menggunakan selector dan property.

Struktur dasar CSS: 
selector {
  property: value;
}

ðŸ“Œ Jenis-Jenis CSS

CSS dibagi menjadi 3 jenis berdasarkan cara penulisannya:

1. Inline CSS

Inline CSS ditulis langsung di dalam tag HTML menggunakan atribut style. Biasanya digunakan hanya untuk satu elemen saja. 

<p style="color: blue; font-size: 20px;">
  Ini paragraf dengan inline CSS
</p>
2. Internal CSS

Internal CSS ditulis di dalam tag <style> pada bagian <head>. Jenis ini cocok jika desain hanya digunakan untuk satu halaman. 

<style>
  h1 {
    color: green;
  }
</style>
3. External CSS

External CSS ditulis pada file terpisah dengan ekstensi .css. Ini adalah cara yang paling direkomendasikan karena rapi dan bisa digunakan banyak halaman. 

<link rel="stylesheet" href="style.css">

Selector adalah cara CSS memilih elemen HTML yang ingin diberi style.

  • Selector Tag: p, h1, div
  • Selector Class: .namaClass
  • Selector ID: #namaId
Contoh selector class: 
.box {
  background-color: yellow;
  padding: 10px;
}

ðŸ“Œ Margin dan Padding

Margin adalah jarak luar elemen, sedangkan padding adalah jarak dalam elemen. Keduanya sangat penting untuk membuat tampilan lebih rapi.

Contoh penggunaan margin dan padding: 
div {
  margin: 20px;
  padding: 15px;
  background-color: white;
}

ðŸ“Œ CSS Layout (Flexbox)

Flexbox adalah salah satu fitur CSS yang berguna untuk mengatur layout elemen agar lebih mudah dan responsif.

Contoh Flexbox:

  display: flex;
  justify-content: center;
  gap: 20px;
}

ðŸ“Œ Pseudo-class (Hover)

Pseudo-class digunakan untuk memberi efek pada kondisi tertentu. Contohnya saat tombol disentuh mouse (hover). 

button:hover {
  background-color: orange;
  color: white;
}

ðŸ“Œ Responsive Design

Responsive design adalah desain website yang dapat menyesuaikan ukuran layar (HP, tablet, laptop). CSS dapat membuat website responsif menggunakan @media.

Contoh media query: 
@media (max-width: 600px) {
  body {
    background-color: lightgray;
  }
}
